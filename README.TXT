Сервис обработки арифметических операций над рациональными числами.

1. используется модуль pyyml
2. запуск скрипта осуществлять из консоли:
    python main.py FILENAME
   , где FILENAME - имя файла с данными (формат json или yaml)
3. примеры файлов с данными: data.json, data.yaml
    Представляют список словарей с ключами/значениями:
        a: str
        b: str (НЕОБЯЗАТЕЛЬНЫЙ)
        operation: str
    , где:
        a и b - рациональное число, представленное в виде дроби ЧИСЛИТЕЛЬ/ЗНАМЕНАТЕЛЬ, каждый из который является целым числом (знаменатель может отсутствовать)
        operation - арифметическая опеация: +, -, *, /
4. результат выполнения: реализация операций и вывод результата
